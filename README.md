# Simple Java Calculator

![Logo Simple Java Calculator](logo.png)

---

This project is developed as part of the Advance Java Technology course. It demonstrates a desktop calculator application using Java Swing for GUI and JDBC for database connectivity. The project aligns with JDBC, focusing on CRUD operations and event-driven GUI design..

---

</div>

This project was **Developed entirely in NetBeans IDE for streamlined GUI design and debugging.**.

If you use the executable "SimpleJavaCalculator.jar" file and that doesn't work, you can type this in your terminal:

```shell
java -jar /your_calculator_directory/SimpleJavaCalculator.jar`
```
